## Base Unit: 4px

All spacing in Prakruti is based on multiples of 4px for consistent alignment.

```

8px  = 0.5rem  = --prakruti-space-2
12px = 0.75rem = --prakruti-space-3
16px = 1rem    = --prakruti-space-4
20px = 1.25rem = --prakruti-space-5
24px = 1.5rem  = --prakruti-space-6
32px = 2rem    = --prakruti-space-8
40px = 2.5rem  = --prakruti-space-10
48px = 3rem    = --prakruti-space-12
64px = 4rem    = --prakruti-space-16
80px = 5rem    = --prakruti-space-20
96px = 6rem    = --prakruti-space-24
```
