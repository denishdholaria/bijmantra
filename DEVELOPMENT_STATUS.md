# Bijmantra Development Status

## 🎯 Project Vision
A modern, BrAPI v2.1 compliant Progressive Web Application for plant breeding management that rivals and exceeds competitors like BreedBase, BMS (Breeding Management System), and GOBII.

│   │   └── store/           # State management

## 🚀 Getting Started

```bash
# Start infrastructure
cd bijmantra
podman compose up -d

# Start backend
cd backend
./start_dev.sh

# Start frontend
cd frontend
npm run dev
```
