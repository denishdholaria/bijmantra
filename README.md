## 🚪 Workspace Gateway

> **New in Dec 2025:** Multi-pathway navigation for diverse user personas

After login, users choose their **workspace** — a personalized pathway through the platform:

| Workspace             | Target Users             | Pages | Key Features                                    |
| --------------------- | ------------------------ | ----- | ----------------------------------------------- |
| 🌾 **Plant Breeding** | Breeders, Geneticists    | 83    | BrAPI v2.1, Genomic Selection, Cross Prediction |
| 🏭 **Seed Business**  | Seed Companies, Labs     | 22    | Quality Testing, Inventory, Dispatch, DUS       |
| 🔬 **Innovation Lab** | Scientists, PhD Students | 28    | AI Vision, WASM Analytics, Space Agriculture    |
| 🏛️ **Gene Bank**      | Curators, Conservation   | 34    | Vault Management, Viability, Germplasm Exchange |
| ⚙️ **Administration** | IT Admins, Managers      | 25    | User Management, System Health, Audit Logs      |

**Features:**

- **Gateway Page** — Full-page workspace selector after login
- **Workspace Switcher** — Quick switch from header dropdown
- **Personalization** — Set default workspace, remembers recent
- **Cross-Access** — Some pages available in multiple workspaces
- **Keyboard Navigation** — Arrow keys, number keys 1-5, Enter to select
- **Accessibility** — Full screen reader support, ARIA labels
