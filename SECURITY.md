## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do not open a public issue.**
2.  Email the project maintainers at denishdholaria@gmail.com with details of the vulnerability.
3.  Include steps to reproduce the issue.
4.  We will acknowledge your report within 48 hours.
5.  We will investigate and provide a timeline for a fix.
6.  Once fixed, we will release a security update and credit you (if desired).

Thank you for helping keep Bijmantra secure!
