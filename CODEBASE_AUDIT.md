## Orphan Pages

The following pages are present in the codebase (`frontend/src/pages/`) but appear to be unreferenced by the active routing configuration (`frontend/src/routes/*.tsx`) or the main application entry point.

*   `frontend/src/pages/Bioinformatics.tsx`
*   `frontend/src/pages/NotFound.tsx`
*   `frontend/src/pages/PerformanceRanking.tsx` (Note: Referenced in detached `plant-sciences` division)
*   `frontend/src/pages/SpatialAnalysis.tsx`

## Orphan Modules

The following modules (divisions) appear to be unreferenced or partially detached from the active routing configuration.

*   `frontend/src/divisions/knowledge`
    *   This module is present on disk but not referenced in `frontend/src/routes/` or `App.tsx`.
*   `frontend/src/divisions/plant-sciences`
    *   The `PlantSciencesHub` page is used in `future.tsx`, but the internal routing definition (`frontend/src/divisions/plant-sciences/routes.tsx`) appears to be unused, superseded by `breeding.tsx`.

## Orphan/Misconfigured Gateways

The following Gateways (Workspaces) have `landingRoute` configurations in `workspaces.ts` that do not match the active routing paths found in `frontend/src/routes/`.

*   **Atmosphere**: Configured `/earth-systems/weather`, but active route appears to be `/weather` or `/sun-earth-systems`.
*   **Hydrology**: Configured `/earth-systems/irrigation`, but active route appears to be `/irrigation` or `/water-irrigation`.
*   **Lithosphere**: Configured `/earth-systems/soil`, but active route appears to be `/soil` or `/soil-nutrients`.
*   **Biosphere**: Configured `/earth-systems/sustainability`, but active route appears to be a placeholder `/sustainability`.
*   **Seed Industry**: Configured `/seed-ops/dashboard`, but typical active routes use `/seed-operations` prefix (e.g., `/seed-operations/samples`).
